# 試験特化レギュレーション判定機

## 説明
試験ごとに異なる特殊レギュレーションにも対応したもの

## 対応ファイル
- 譜面ファイル
  - [ ] `.sus` 
  - [x] `.usc`

- その他
  - [ ] `.json`
  - [ ] `.txt`


## 使い方
1. `.usc`ファイルを選択する
2. おしまい


> [!CAUTION]
> ファイルの容量によっては結果の表示までに時間がかかる場合があります
