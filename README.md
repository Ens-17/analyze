# 試験特化レギュレーション判定機

## 説明
試験ごとに異なる特殊レギュレーションにも対応し、違反項目を少し細分化したもの

## 対応ファイル
- 譜面ファイル
  - [ ] `.sus` 
  - [x] `.usc`

- その他
  - [ ] `.json`
  - [ ] `.txt`

## 使い方
1. `.usc`ファイルを選択する
2. おしまい

## その他の細かい説明
1. ️⭕️と❌について
     - ️⭕️は「公式レギュでは無いが今回の試験では使用可能なもの」
     - ❌は「シンプルにレギュ違反」<br><br>
2. [　　]に囲まれた数字について
     - その小節のどこかに違反している箇所があります。小節数はMMWを基準にし、最初を0小節目としてカウントしています。<br><br>
3. 同じ項目が二つ以上引っかかった場合について
     - どちらかひとつしか出ません。
     - 「どこが違反しているか」より「そもそも違反しているかどうか」を重視しています

<br>

> [!CAUTION]
> ファイルの容量によっては結果の表示までに時間がかかる場合があります
>
