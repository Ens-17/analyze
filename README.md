# 試験特化レギュレーション判定機

## 説明
試験ごとに異なる特殊レギュレーションにも対応したもの

## 対応ファイル
- 譜面ファイル
  - [ ] `.sus` 
  - [x] `.usc`

- その他
  - [ ] `.json`
  - [ ] `.txt`

## 使い方
1. `.usc`ファイルを選択する
2. おしまい

## その他の細かい説明
1. ️⭕️と❌について
     - ️⭕️は「公式レギュでは無いが今回の試験では使用可能なもの」
     - ❌は「シンプルにレギュ違反」<br><br>
2. [　　]に囲まれた数字について
     - そのusc内で引っかかった行数を表示しています

<br>

> [!CAUTION]
> ファイルの容量によっては結果の表示までに時間がかかる場合があります
> 
<br>
